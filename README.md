# 📊 Analyse des performances d’un hôtel à N’Djamena

## 🎯 Objectif
Ce projet vise à analyser les performances d’un hôtel afin d’identifier des axes d’amélioration des revenus et de réduction des pertes.

---

## 🧩 Données
Le dataset contient :
- Dates de réservation  
- Type de chambre  
- Prix par nuit  
- Nombre de nuits  
- Type de client  
- Canal de réservation  
- Statut (Confirmé / Annulé)  
- Revenus  

---

## 📈 Outils utilisés
- Excel  
- Power BI  

---

## 🔍 Analyses réalisées
- Évolution des revenus mensuels  
- Revenus par type de chambre  
- Revenus par canal  
- Analyse des clients  
- Taux d’annulation  

---

## 💡 Insights
- Certaines chambres génèrent plus de revenus  
- Forte dépendance à certains canaux  
- Périodes avec forte annulation  

---

## 🚀 Recommandations
- Développer la vente directe  
- Optimiser les périodes creuses  
- Réduire les annulations
